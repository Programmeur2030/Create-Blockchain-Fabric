# Create Blockchain (Fabric)

This is a Fabric mod that integrates blockchain mechanics into the Create mod ecosystem.

## Installation

1. Clone the repository.
2. Open the project in your IDE (IntelliJ IDEA recommended).
3. Run `gradlew genSources` to generate Minecraft sources.
4. Run the `Fabric Client` run configuration.

## Dependencies

- Minecraft 1.20.1
- Fabric Loader
- Fabric API
- Create (Fabric)
- Create Crafts & Additions (Fabric)
- Create Numismatics (Fabric)
